### Reflection
### What challenges did you face when refactoring your code to use Bootstrap?
I have looked into traditional CSS here and there throughout my life and was therefore familiar with it but it was a challenge for me to get over the fear of using something completely new like bootstrap. The official documentation site was thankfully easy to digest and therefore made learning much easier.
### How did using Bootstrap utility classes and components simplify your styling process?
Not having to switch back and forth between HTML and CSS was a timesaver. Bootstrap utilies and components are pre-defined and consistent in their behavior which makes the website very uniform.
### In what scenarios might you choose not to use Bootstrap and write custom CSS instead?
It may make more sense to utilize custom CSS when there is a very specific look and feel that has to be created where Bootstrap may fall short. There may also be performance concerns with loading a large library like Bootstrap.

### What I learned
Bootstrap is great and has more to offer than what I explored via this assignment. I do find myself more comfortable with traditional CSS since that is what I have been familiar with but I believe strongly that once I have a better grasp of the syntax, my preference will change.

### Continued development
I see myself using Bootstrap more in the future for practice.
